Theme Nova

Premium Modern Backend Theme for Odoo 16

Transform your Odoo backend into a world-class SaaS experience. Inspired by Linear, Notion, and Stripe — designed for professionals who demand excellence.

Key Features

iOS-Style App Launcher

Beautiful grid of colorful app icons with search, hover effects, and staggered animations.

Dark & Light Mode

Toggle between modes with a single click. System preference auto-detection supported.

Glassmorphism Navbar

Refined, compact navbar with blur effect, elegant dropdowns, and clean systray icons.

Collapsible Sidebar

Smart sidebar navigation that collapses gracefully with smooth animations.

Redesigned Views

List, Kanban, Form, Calendar, Pivot, and Graph views — all modernized.

Micro-Interactions

Subtle animations, skeleton loaders, and smooth transitions throughout.

Configuration

Go to Settings > Theme Nova to customize: accent color (6 presets), dark/light mode, animations toggle, sidebar default state.
